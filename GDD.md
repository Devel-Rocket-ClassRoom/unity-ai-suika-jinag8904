# Game Design Document — 수박 게임 (Suika Game)

> Unity 6 · URP 2D  
> 작성일: 2026-05-12

---

## 1. 게임 개요

**한 줄 요약**: 과일을 떨어뜨려 같은 과일끼리 합쳐 더 큰 과일을 만들고, 상자가 넘치기 전에 최고 점수를 노리는 물리 기반 낙하 퍼즐.

**코어 루프**:
```
과일 선택 → X 위치 조정 → 낙하 → 물리 안착
→ 동일 과일 충돌 → 합성(다음 단계 과일 생성) → 점수 획득
→ 다음 과일 선택 … (상자 넘침 → 게임 오버)
```

---

## 2. 핵심 메커닉

### 떨어뜨리기
- 과일 하나가 컨테이너 상단에 나타난다. 플레이어는 **좌우 위치만** 조정 가능.
- 입력(클릭·탭)으로 과일을 낙하시킨다.
- 낙하 후 약 **1.2초 쿨타임** — 물리가 안착할 때까지 다음 과일 스폰 대기.
- 다음에 올 과일은 **넥스트 프리뷰**로 1개 미리 공개.
- 스폰 가능 과일은 **1~5 티어(체리~감)**만 — 더 큰 과일은 합성으로만 등장.

### 충돌 (물리)
- 모든 과일은 **원형 콜라이더 + Rigidbody2D(Dynamic)** 구성.
- 중력의 영향을 받아 자유 낙하하며, 착지 후 굴러 다른 과일 위에 쌓인다.
- 작은 과일 → 반발력 높고 가볍다 / 큰 과일 → 무겁고 덜 튄다.
- 합성 폭발력이 인접 과일을 튕겨내어 **연쇄 반응(콤보)** 또는 위험 상황을 만들 수 있다.

### 머지 (합성)
- 동일 티어 과일 두 개가 접촉하면 즉시 합성 발동.
- 두 과일이 파괴되고 **중간 지점에 다음 티어 과일**이 생성된다.
- **이중 합성 방지**: 이미 합성 중인 과일은 재합성되지 않는다.
- **수박(11 티어) 합성** — 두 수박이 모두 소멸하고 추가 보너스 점수 지급. 새 과일 스폰 없음.
- 합성이 연속으로 터지는 **체인 콤보**는 주요 고득점 수단.

### 게임 오버
- 컨테이너 상단에 **데스라인**이 존재한다.
- 어떤 과일이든 데스라인 위에 **약 0.8초 이상** 머무르면 게임 오버.
  - 낙하·반등으로 순간 접촉하는 경우는 게임 오버가 아님.
- 시간 제한 없음 — 순수 공간 관리 퍼즐.

## 3. 과일 단계

총 **11종**. 체리가 가장 작고, 수박이 가장 크다.

| 티어 | 과일 | 합성 점수 | 반지름 비율 | 스폰 가능 |
|:----:|------|:---------:|:-----------:|:---------:|
| 1 | 체리 (Cherry) | 1 | 1.0× | ✓ |
| 2 | 딸기 (Strawberry) | 3 | 1.3× | ✓ |
| 3 | 포도 (Grape) | 6 | 1.7× | ✓ |
| 4 | 데코폰 (Dekopon) | 10 | 2.2× | ✓ |
| 5 | 감 (Persimmon) | 15 | 2.8× | ✓ |
| 6 | 사과 (Apple) | 21 | 3.5× | — |
| 7 | 배 (Pear) | 28 | 4.0× | — |
| 8 | 복숭아 (Peach) | 36 | 4.8× | — |
| 9 | 파인애플 (Pineapple) | 45 | 5.5× | — |
| 10 | 멜론 (Melon) | 55 | 6.5× | — |
| 11 | 수박 (Watermelon) | 66 + 보너스 81 | 8.0× | — |

> 체리 반지름을 기준 단위(1.0)로 설정. 수박 직경 ≈ 컨테이너 폭의 절반.  
> 합성 순서: 체리 → 딸기 → 포도 → 데코폰 → 감 → 사과 → 배 → 복숭아 → 파인애플 → 멜론 → 수박 → (소멸)

## 4. 점수 · 승패 규칙

### 점수
- 합성 발생 시 **결과 과일의 점수값**만큼 즉시 추가.
  - 예) 체리 × 2 → 딸기 합성 = **3점**
- 수박 합성 보너스: **81점** 추가 지급.
- 개인 최고 점수 로컬 저장.

| 수준 | 기준 점수 |
|------|-----------|
| 입문 | 1,000~1,500 |
| 숙련 | 3,000 |
| 고수 | 5,000+ |

### 승패
- **승리 조건 없음** — 끝없이 이어지는 스코어 어택.
- 커뮤니티 기준 "클리어" = 수박 두 개 합성 성공(소멸).
- **패배** = 데스라인 초과 (게임 오버).

## 5. 화면 · UI 구성

```
┌──────────────────────────────┐
│ [점수]            [넥스트]   │  ← 상단 HUD
│                              │
│         ▼ (낙하 과일)        │  ← 컨테이너 위, 플레이어 조작
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← 데스라인
│ │                          │ │
│ │                          │ │
│ │   (쌓여가는 과일들)       │ │  ← 컨테이너
│ │                          │ │
│ └──────────────────────────┘ │
│                   [진화표]   │  ← 우측 하단
└──────────────────────────────┘
```

| UI 요소 | 위치 | 내용 |
|---------|------|------|
| 현재 점수 | 상단 좌측 | 합성마다 실시간 갱신 |
| 최고 점수 | 점수 아래 | 로컬 최고 기록 |
| 넥스트 프리뷰 | 상단 우측 | 다음 낙하 과일 1개 예고 |
| 낙하 과일 | 컨테이너 상단 중앙 | 현재 들고 있는 과일, X 이동 가능 |
| 데스라인 | 컨테이너 상단 가로선 | 시각적 경계선 |
| 진화 차트 | 우측 하단 | 11단계 진화 순서 참고용 |
| 게임 오버 팝업 | 화면 중앙 | 최종 점수 + 최고 점수 + 재시작 버튼 |

## 6. 톤 & 분위기

### 색감
- **배경**: 크림·베이지 계열의 따뜻하고 밝은 단색 (눈에 거슬리지 않게).
- **컨테이너**: 나무 상자 — 따뜻한 골드·앰버 톤 목재 질감.
- **과일**: 채도 높고 선명한 원색. 각 과일이 색으로 즉시 구분 가능해야 함.

### 아트 방향
- 귀엽고 친근한 2D 카툰 스타일. 남녀노소 부담 없는 캐주얼 분위기.
- 과일은 **둥근 얼굴 표정** 포함 — 평상시 웃음, 데스라인 근처에서 찡그림.
- 합성 시 짧은 **스케일 팝 애니메이션** + 파티클 이펙트로 쾌감 강조.
- 전체 비주얼은 단순하고 깔끔하게 — 과일이 주인공, 배경은 서포터.

### BGM · 사운드
- BGM: 루핑되는 가볍고 경쾌한 캐주얼 음악. 반복 플레이에 피로감이 없어야 함.
- 합성 SFX: 뽀각·팝 계열의 짧고 만족스러운 효과음.
- 게임 오버 SFX: 짧은 실패 징글 (과하지 않게).
- 수박 합성: 특별한 트리엄프 사운드로 차별화.
- 전체 사운드 톤: 밝고 가볍게, 스트레스 없이.
